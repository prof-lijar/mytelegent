# Tiny Jarvis Vision

## Mission Statement
Tiny Jarvis is a local personal AI agent that empowers users to schedule and automate Telegram messages with natural language commands, while maintaining strict safety controls.

## Key Features
1. Natural language parsing of scheduling commands
2. Local LLM processing (Gemma via Ollama)
3. Secure message scheduling with retry logic
4. Telegram userbot integration with random send delays
5. Full local operation with no cloud dependency

## Target Audience
- Busy professionals needing task automation
- Privacy-conscious users
- Developers looking for a local AI agent example

## Safety Commitment
- No mass messaging
- No message logging of content
- Max 2 retries per message
- Explicit user confirmation required
